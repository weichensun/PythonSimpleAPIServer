## About

A simple http server framework base on Python BaseHTTPServer

#### Features
	Support python2 and python3
	Use threading to process requests (No blocking)
	Support https
	Include simple route
	Dynamic load worker class

#### Support methods
	GET	-
	POST	-
	PUT	-
	DELETE	-
